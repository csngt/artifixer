#  Artifixer – AI Content Detection & Humanization System

Artifixer is a Django-based web application that detects AI-generated text and images and provides humanization features to refine AI-written content.

It combines Machine Learning models with a user-friendly web interface.

---

##  Features

### 🔹 AI Text Detection
- Uses BERT Tokenizer
- Random Forest Classifier
- Detects whether text is AI-generated or human-written

### 🔹 AI Image Detection
- CNN-based image classification model
- Detects AI-generated images

### 🔹 Text Humanization
- Uses NLP techniques (NLTK, WordNet)
- Rewrites AI-generated text to sound more natural

### 🔹 Web Interface
- Built using Django
- Clean Bootstrap-based frontend
- User-friendly UI
